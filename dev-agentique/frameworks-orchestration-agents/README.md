# Agent Orchestration Frameworks — LangGraph, LangChain, AutoGen

> **Language:** This course is written in **French**.

A comprehensive self-learning course on the main agent orchestration frameworks used in production in 2026.

## What's covered

| Module | Topics |
|--------|--------|
| 1 — Fundamentals | Agent loop, what orchestration means, 2026 landscape, how to choose |
| 2 — LangChain | LCEL & Runnable, models/parsers, tool calling, RAG, common pitfalls |
| 3 — LangGraph | StateGraph, conditional edges, ReAct loop, checkpointing, human-in-the-loop, multi-agents, streaming |
| 4 — AutoGen / AG2 | Conversational paradigm, AssistantAgent, GroupChat, async architecture, AG2 vs Microsoft Agent Framework |
| 5 — Multi-agent patterns | Supervisor, pipeline, fan-out/fan-in, shared memory, MCP, nested agents |
| 6 — Production | Error handling & retries, observability, cost control, security, deployment |
| 7 — Comparison | Decision table, same agent built in all 3 frameworks side by side, anti-patterns |

## How to open
