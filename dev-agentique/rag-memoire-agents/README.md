# RAG et mémoire pour agents — Vector Stores, ChromaDB

> **Language:** This course is written in **French**.

A comprehensive self-learning course on Retrieval-Augmented Generation (RAG) and agent memory systems, with a focus on ChromaDB and production best practices as of 2026.

## What's covered

| Module | Topics |
|--------|--------|
| 1 — RAG Fundamentals | Why RAG, end-to-end architecture, embeddings, vector similarity |
| 2 — Vector Stores | What is a vector store, ChromaDB architecture, alternatives (FAISS, Qdrant, Pinecone), how to choose |
| 3 — ChromaDB in practice | Collections, documents, metadata, queries & filters, deployment modes, CRUD |
| 4 — Chunking & indexation | Chunking strategies, chunk size/overlap impact, metadata enrichment, incremental indexing |
| 5 — Advanced retrieval | Hybrid search (dense + BM25), reranking with cross-encoders, MMR diversity, query transformation & HyDE |
| 6 — Agent memory | 4 types of memory, short-term context management, long-term vector memory, LangGraph Store |
| 7 — Agentic RAG | From linear RAG to retrieval agent, Self-RAG, Corrective RAG, multi-agent routing |
| 8 — Evaluation & production | RAGAS metrics, debugging failures, performance, security (RAG poisoning), scaling |

## How to open

Open `index.html` directly in **Chrome**. No server or installation needed.
